A set of components and utilities for creating simple websites.
Built using SvelteKit

Import components from "hd-web/components" and utilities from "hd-web/utils"

For Typescript to work, add moduleResolution: "Bundler" to tsconfig.json
(See https://kit.svelte.dev/docs/packaging#typescript)

You can overwrite the css variables used in this project via the cssVariablesPlugin for vite
